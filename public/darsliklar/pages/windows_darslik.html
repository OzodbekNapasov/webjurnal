Windows Operatsion Tizimi va Texnik Xizmat Ko'rsatish

"Tibbiyotda axborot texnologiyalari" fani talabalari uchun maxsus tayyorlangan elektron darslik. Windows OT tarixi, ish stoli va fayllar bilan ishlash asoslari, tizimga qo'shimcha xizmatchi dasturlar o'rnatish hamda kompyuterga texnik xizmat ko'rsatishni interaktiv darslar, simulyatorlar va testlar orqali mukammal o'rganing.

Darsni Boshlash

1-Bo'lim. Kirish: Operatsion Tizimlar

Operatsion tizim nima, uning kompyuterdagi vazifasi va Windows tizimining tutgan o'rni.

Operatsion tizim nima?

Operatsion tizim (OT) — kompyuterning barcha apparat qismlarini (protsessor, xotira, qattiq disk) boshqaruvchi hamda foydalanuvchining dasturlar bilan aloqasini (interfeys) ta'minlovchi eng asosiy tizimli dasturiy ta'minotdir. Tizim yordamida apparatlar "jonlanadi".

OTning asosiy vazifalari

Fayllar tizimini boshqarish, xotirani taqsimlash, jarayonlarni parallel boshqarish (ko'p vazifalilik), kompyuter xavfsizligini ta'minlash, ulangan tashqi qurilmalarni (printer, skaner, tibbiy UZI apparatlari) drayverlar orqali OT bilan bog'lash.

Windows tizimining o'rni

Microsoft Windows — jahondagi shaxsiy kompyuterlarning 70% dan ortig'iga o'rnatilgan eng mashhur operatsion tizimdir. O'zining grafik interfeysi (GUI), foydalanish uchun qulayligi va tibbiy axborot tizimlari uchun mosligi bilan tibbiyot sohasida ham mutlaq yetakchidir.

2-Bo'lim. Windows Operatsion Tizimi Tarixi

MS-DOS dan to Windows 11 gacha bo'lgan tarixiy rivojlanish bosqichlari bilan interaktiv tarzda tanishing.

MS-DOS (1981)

1981

Asosiy imkoniyatlari: Matnli konsol (CLI) orqali buyruqlar kiritish, bitta vazifali (single-tasking) disk operatsion tizimi.

Yangiliklari: IBM PC shaxsiy kompyuterlarining dastlabki standart tizimi hisoblanadi. Unda grafik interfeys mavjud bo'lmagan, barcha amallar yozma buyruqlar orqali amalga oshirilgan.

Windows 1.0 (1985)

1985

Asosiy imkoniyatlari: Sichqoncha orqali boshqariladigan grafik interfeysning (GUI) ilk bor taqdim etilishi.

Yangiliklari: MS-DOS ustiga o'rnatiladigan grafik dastur qobig'i bo'lgan. Clipboard (nusxa olish xotirasi), bloknot, taqvim, kalkulyator va soat kabi dasturlar yaratildi.

Windows 3.1 (1992)

1992

Asosiy imkoniyatlari: TrueType vektor shriftlarini qo'llab-quvvatlash, 16-bitli multimedia va audio kengaytmalari.

Yangiliklari: Virtual xotirani boshqarish tizimi joriy qilindi. Mashhur "Mina qidiruvchisi" (Сапер) o'yini va Program Manager oynasi paydo bo'ldi.

Windows 95 (1995)

1995

Asosiy imkoniyatlari: To'liq 32-bitli arxitektura, Plug and Play drayverlar tizimi, uzun fayl nomlarini yozish imkoniyati.

Yangiliklari: Birinchi marta zamonaviy Windowsning "yuragi" bo'lmish Рабочий стол, Кнопка Пуск va Панель задач joriy qilindi. Internet Explorer brauzeri qo'shildi.

Windows 98 (1998)

1998

Asosiy imkoniyatlari: USB portlari va flesh-xotiralarni to'liq qo'llab-quvvatlash, FAT32 fayl tizimi.

Yangiliklari: Tizimning internet bilan integratsiyasi kuchaytirildi, barqarorlik oshirildi, "Проводник" boshqaruvi takomillashtirildi.

Windows XP (2001)

2001

Asosiy imkoniyatlari: Professional darajada barqaror ishlaydigan Windows NT yadrosi, ko'p foydalanuvchili tizim.

Yangiliklari: Yangi "Luna" dizayni, CD yozish imkoniyati. Tizim favqulodda barqarorligi sababli tibbiyot uskunalarida (rentgen, kardiograf, UZI) uzoq yillar yetakchi bo'lib keldi.

Windows Vista (2007)

2007

Asosiy imkoniyatlari: Windows Aero vizual shaffof interfeysi, tezkor qidiruv, kuchaytirilgan xavfsizlik (UAC).

Yangiliklari: "Защитник Windows" (Windows Defender) antivirusi va diskni BitLocker shifrlash tizimlari ilk marta taqdim qilindi.

Windows 7 (2009)

2009

Asosiy imkoniyatlari: Resurslarni juda tejamkor boshqarish, tezkor yuklanish, sensorli ekranlarni qo'llab-quvvatlash.

Yangiliklari: Oynalarni tartibga soluvchi Aero Snap, tozalangan panel va tibbiyot muassasalarida eng uzoq foydalanilgan xavfsiz tizimlardan biri.

Windows 8 / 8.1 (2012)

2012

Asosiy imkoniyatlari: Planshetlar va sensorli monitorlarga moslashtirilgan "Metro UI" plitkali interfeysi.

Yangiliklari: "Кнопка Пуск" butunlay olib tashlandi (bu foydalanuvchilar e'tiroziga sabab bo'lib, 8.1 da qaytarildi), Windows App Store do'koni yaratildi.

Windows 10 (2015)

2015

Asosiy imkoniyatlari: Gibrid boshqaruv (planshet va kompyuter), virtual ish stollari, Microsoft Edge brauzeri.

Yangiliklari: Microsoft Windowsni doimiy ravishda internet orqali yangilanib boruvchi xizmat (Windows as a Service) deb e'lon qildi. Shifoxonalardagi barcha zamonaviy kompyuterlarning standart operatsion tizimi.

Windows 11 (2021)

2021

Asosiy imkoniyatlari: Markazlashgan "Кнопка Пуск", yumaloq oynalar, o'rnatilgan Microsoft Teams tizimi.

Yangiliklari: Ishlash samaradorligi va kiberxavfsizlik (TPM 2.0 talabi) sezilarli darajada oshirildi. Tibbiy tashxis dasturlari va ma'lumotlar xavfsizligini ta'minlash uchun mo'ljallangan.

Bosqichli Amaliy Ko'rsatma

1. Papka va fayl yaratish:

Bo'sh joyda sichqoncha o'ng tugmasini bosib, Создать -> Папку yoki Создать файлni tanlang.

2. Nusxalash va ko'chirish:

Faylni tanlab o'ng tugmani bosing va Копировать (nusxalash) yoki Вырезать (qirqib olish) ni bosing, so'ngra bo'sh joyga o'ng bosib Вставитьni bosing.

3. Qayta nomlash va o'chirish:

Tegishli element ustida o'ng tugma -> Переименовать (nomini o'zgartirish) yoki Удалить (savatchaga tashlash) ni bosing.

5-Bo'lim. Qo'shimcha Xizmatchi Dasturlar

Operatsion tizimni foydali xizmatlar bilan boyituvchi asosiy yordamchi dasturlar.

WinRAR

Vazifasi: Katta o'lchamdagi fayl va papkalarni siqish, arxivlash hamda himoya qilish.

Afzalliklari: Ko'plab fayllarni yagona arxiv faylga jamlab arxiv hajmini kichraytiradi.

Tibbiyotda: Rentgen rasmlari va kasallik tarixi arxivlarini siqib shifoxona pochtasida yuborish.

7-Zip

Vazifasi: Yuqori darajadagi siqish tezligiga ega bo'lgan ochiq kodli va butunlay bepul arxivator.

Afzalliklari: O'zining 7z yuqori siquvchi formatiga ega va tizim resurslarini juda kam egallaydi.

Tibbiyotda: Davlat shifoxonalarida litsenziya xarajatlarisiz bepul ofis arxivatori sifatida foydalanish.

Adobe Reader

Vazifasi: Standard PDF formatidagi hujjatlarni ko'rish, sharhlash va chop etish.

Afzalliklari: Hujjatni butun dunyoda bir xil ko'rinishda saqlaydi, matnlarga sharh va imzo kiritish imkoni bor.

Tibbiyotda: MRT, UZI xulosalarini va xalqaro klinik protokollarni ochish.

Google Chrome

Vazifasi: Jahondagi eng tezkor va xavfsiz internet brauzeri.

Afzalliklari: Bulutli ma'lumotlarni sinxronlash va kengaytmalarni qo'llab-quvvatlash tizimi.

Tibbiyotda: Bulutli shifoxona axborot tizimlariga (HIS) va Medline ilmiy bazalariga kirish.

VLC Media Player

Vazifasi: Har qanday formatdagi multimedia fayllarini ochuvchi bepul pleyer.

Afzalliklari: Balla kodeklar o'rnatilgan, videoni kuchaytirib va sifatli ijro etib beradi.

Tibbiyotda: Tibbiy endoskopiya yoki jarrohlik amaliyotlarining video tasvirlarini tahlil qilish.

Kaspersky

Vazifasi: Kompyuterni viruslar, troyanlar va xakerlik hujumlaridan himoya qilish.

Afzalliklari: Real vaqt rejimida himoya devori (firewall) va bulutli kiber-monitoring.

Tibbiyotda: Bemorlar bazasi va shaxsiy ma'lumotlarni maxfiy saqlash hamda himoyalash.

Очистка диска (Disk Cleanup)

Vazifasi:

Qattiq diskda to'plangan vaqtinchalik keraksiz fayllar (Temporary files), keshlar va savatcha ichidagi fayllarni tozalab, tizimda bo'sh joy ochish hamda tezlikni oshirish.

Foydalanish tartibi

Пуск -> "Очистка диска" yoziladi -> Disk tanlanib keraksiz fayl turlari belgilanadi -> OK tugmasi bosiladi.

Afzalliklari

Diskdagi ortiqcha yuzlab megabayt keraksiz axlat fayllarni xavfsiz tozalaydi, tizim barqarorligini oshiradi.

Tibbiyot axborot texnologiyalarida qo'llanilishi:

Shifoxona kompyuterlarida katta hajmdagi bemor rentgen va MRT tasvirlari saqlanishi tufayli yuzaga keladigan disk to'lish muammosini xavfsiz tozalash orqali bartaraf etish.

8-Bo'lim. Kompyuterni Parvarish Qilish

Kompyuterning fizikaviy va dasturiy barqarorligini asrash bo'yicha interaktiv tavsiyalar (Kartochkalarni bosing).

Changdan tozalash
Batafsil

Kompyuter ichidagi chang ventilyatorlarni (kulerlarni) qotiradi va qizishga olib keladi. Kompyuterni yilda kamida 1-2 marta profilaktik changdan tozalash kerak.

Elektr xavfsizligi
Batafsil

Tok urishi yoki kuchlanish o'zgarishidan asrash uchun shifoxonalarda UPS (Uzluksiz quvvat manbai) bilan ishlash shart. Bu tibbiy apparatlarni saqlaydi.

Antivirus ishlatish
Batafsil

Begona USB fleshkalarni ochishdan oldin virusga tekshirish kerak. Litsenziyali antivirus yoki Windows Defender doimiy real vaqtda ishlab turishi lozim.

Kuchli parol yaratish
Batafsil

Bemorlar ma'lumotlarini himoyalash uchun parolda kamida 8 ta belgi, bosh va kichik harflar, raqamlar va belgilar (@, #, $) bo'lishini ta'minlang.

Ma'lumotlarni himoyalash
Batafsil

Muhim shaxsiy va tibbiy fayllarni shifrlash uchun Windows BitLocker texnologiyasidan foydalanish hamda maxfiy hujjatlarni doimiy yashirin saqlash.

To'g'ri o'chirish
Batafsil

Kompyuterni bevosita tokdan sug'urmasdan, har doim Пуск -> Завершение работы orqali o'chirish lozim. Bu qattiq disk fayl tizimini asraydi.

9-Bo'lim. Amaliy Topsiriqlar

Mavzuni amalda o'zlashtirish uchun quyidagi topshiriqlarni bajaring va ularni belgilab boring.

Mustaqil amaliy topshiriqlar 0% Bajarildi

Windows 10/11 ish stolida yangi papka yaratish (Создать -> Папку)
Papka ichida yangi matnli hujjat yaratish (Создать -> Текстовый документ)
Yaratilgan faylni nusxalab (Копировать -> Вставить) nusxa hosil qilish
Fayl nomini o'zgartirish (Переименовать -> "Epicrisis")
Windows darslik simulyatorida WinRAR dasturini o'rnatish
"Очистка диска" (Disk Cleanup) yordamchi dasturini ishga tushirib diskni tozalash tartibini o'rganish
"Центр обновления Windows" (Update Center) orqali yangilanishlarni tekshirish amalini bilish

11-Bo'lim. Yakuniy Xulosa

Windows operatsion tizimini o'rganishning kelajak shifokorlari faoliyatidagi ahamiyati.

Tibbiyot va Tizimli Texnologiyalar Integratsiyasi

Zamonaviy shifoxonalar, laboratoriyalar va diagnostika markazlari (MRT, raqamli rentgen, MSKT) butunlay shaxsiy kompyuterlar va Windows operatsion tizimiga tayanadi. Kelajak shifokorlarining operatsion tizimda erkin ishlashi, fayllarni xavfsiz boshqarishi, dasturlarni to'g'ri o'rnatishi va tizimni texnik parvarishlay olishi — tibbiy yozishmalar va ma'lumotlar xavfsizligini ta'minlashning eng muhim shartlaridan biridir.

Ushbu darslik tibbiyot sohasida faoliyat yurituvchi mutaxassislarga raqamli ko'nikmalarni oshirishda poydevor bo'lib xizmat qiladi.

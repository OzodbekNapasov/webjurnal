1-Bo'lim. Kirish

Zamonaviy dunyoda Internet barcha sohalar kabi tibbiyotda ham asosiy boshqaruv va aloqa vositasiga aylandi. Internet — bu butun dunyodagi millionlab kompyuter va tarmoqlarni o'zaro bog'lovchi global axborot tizimidir. U ma'lumotlarni paketlar shaklida uzatish (IP protokoli) yordamida ishlaydi.

Tarmoq nima?

Ikki yoki undan ortiq qurilmalarning ma'lumot almashish maqsadida o'zaro simli (kabel) yoki simsiz (Wi-Fi) bog'lanishi tarmoq deb ataladi.

Internet nima uchun kerak?

Masofaviy ma'lumotlar omboriga kirish, tezkor xabar almashish, ilmiy manbalardan foydalanish hamda avtomatlashtirilgan xizmatlarni yo'lga qo'yish.

Tibbiyotdagi ahamiyati:

Tibbiyot oliygohlari talabalari uchun tarmoq texnologiyalarini bilish quyidagi zamonaviy tizimlar bilan ishlashda o'ta muhim hisoblanadi:

  • Elektron tibbiy kartalar (E-Health / EHR): Bemorlar tarixi bulutli ma'lumotlar omborida saqlanadi va internet orqali real vaqtda yangilanadi.
  • Teletibbiyot (Telemedicine): Shifokorlar uzoq hududlardagi bemorlarni internet orqali video-konsultatsiya qiladilar hamda jarrohlik operatsiyalarini masofadan kuzatadilar.
  • Tibbiy axborot tizimlari (MIS): Poliklinika va shifoxonalar ma'lumotlarini boshqarish tizimlari butunlay internet ulanishiga tayanadi.
  • Elektron pochta va xabarlar: Shifoxonalar o'rtasida tahlil natijalari va rentgen/KT tasvirlarini yuborish.

4-Bo'lim. Noutbukni Wi-Fi ga ulash

Noutbuklar ichki Wi-Fi modulga ega bo'lganligi sababli qo'shimcha qurilmasiz ulanadi. Windows operatsion tizimining rus tilidagi interfeysi misolida bosqichma-bosqich jarayonni ko'ring:

Ruscha sozlash ketma-ketligi:
  1. Ekranning pastki o'ng burchagida joylashgan Панель задач (Vazifalar paneli) tizim paneli (tray) hududiga o'ting.
  2. Tarmoq belgisi — Значок Wi-Fi / Сеть ustiga bosing.
  3. Ochilgan oynada faol simsiz tarmoqlar ro'yxatidan keraklisini tanlang.
  4. Подключиться (Ulanish) tugmasini bosing.
  5. Agar tarmoq himoyalangan bo'lsa, xavfsizlik kalitini (parol) kiriting va Далее (Keyingi) tugmasini bosing.

5-Bo'lim. Shaxsiy kompyuterni Wi-Fi ga ulash

Aksariyat statsionar shaxsiy kompyuterlar (PC) ona platasida ichki Wi-Fi modul bo'lmaydi. Ularni simsiz tarmoqqa ulash uchun Wi-Fi adapterlaridan foydalaniladi:

USB Wi-Fi adapter

Kompyuterning istalgan USB portiga o'rnatiladigan kichik qurilma. O'rnatish juda oson (Plug & Play), lekin tezligi PCIe adapterlariga qaraganda biroz pastroq bo'lishi mumkin.

PCIe Wi-Fi adapter

Tizim bloki ichiga, ona plataning PCI Express slotiga o'rnatiladigan tarmoq platasi. Tashqi antennalarga ega bo'lib, tezroq va barqarorroq uzatish tezligini beradi.

6-Bo'lim. Ethernet kabel orqali ulanish

Ethernet (kabel) orqali ulanish eng tez va barqaror ulanish turi hisoblanadi. Buning uchun RJ-45 konnektorli tarmoq kabeli va kompyuterdagi LAN porti kerak bo'ladi.

Ulanish bosqichlari:
  • Kabelning bir uchini Router orqa panelidagi LAN portiga, ikkinchi uchini esa shaxsiy kompyuter orqasidagi LAN portiga ulaymiz.
  • Windows sozlamalarida ulanishni tekshirish uchun: Параметры → Сеть и Интернет → Ethernet menyusiga o'tamiz.
  • Agar ulanish to'g'ri bo'lsa, tarmoq holati "Сеть" (yoki tarmoq nomi) va "Подключено" rejimida ko'rinadi.

7-Bo'lim. Telefon hotspoti orqali internetga ulanish

Tashqarida yoki uydagi router o'chib qolganda smartfon mobil internetini kompyuterga ulash mumkin:

Android telefonlarida:

Настройки → Сеть и Интернет → Точка доступа и модем → Мобильная точка доступа menyusiga kirib, faollashtiring. Shundan so'ng, noutbuk Wi-Fi ro'yxatida telefon nomi va paroli paydo bo'ladi.

iPhone telefonlarida:

Настройки → Режим модема bo'limiga kirib, "Разрешать другим" tugmasini yoqing. Wi-Fi orqali ulanish rejimiga o'tadi.

Telefonga kompyuterni ulash usullari:
  • Wi-Fi modem: Telefonni simsiz router sifatida ishlatish (eng keng tarqalgan).
  • USB modem: Telefonni USB kabel orqali kompyuterga ulab, sozlamalardan "USB-модем" rejimini yoqish. statsionar PC uchun juda mos.
  • Bluetooth modem: Bluetooth orqali ulash (tezligi sekinroq, lekin energiya kam sarflaydi).

9-Bo'lim. IP manzil va DNS

Har bir tarmoqqa ulangan qurilmaning o'zining yagona manzili bo'ladi. Ular IP-manzil va DNS yordamida muloqot qiladi:

  • IP address (Internet Protocol Address): Tarmoqdagi qurilmaning raqamli manzili.
    • IPv4: To'rt qismdan iborat raqamlar, masalan: 192.168.1.34. Router IP manzili ko'pincha 192.168.1.1 bo'ladi.
    • IPv6: Mukammalroq va kengroq harfli-raqamli manzil.
  • DNS (Domain Name System): Sayt nomlarini IP manzillariga aylantirib beruvchi tarmoq xizmati. Masalan, siz brauzerda google.com deb yozganingizda, DNS uni 142.250.190.46 raqamiga o'zgartiradi.
    • Mashhur ochiq DNS: 8.8.8.8 (Google DNS), 1.1.1.1 (Cloudflare).

11-Bo'lim. Tarmoq muammolarini aniqlash

Internet ishlamay qolganda yoki uzilishlar bo'lganda, quyidagi amallarni ketma-ket bajarish tavsiya qilinadi:

Diagnostic bosqichlari (Ruscha):
  1. O'ng pastki tarmoq belgisini sichqoncha o'ng tugmasi bilan bosib, Диагностика неполадок (Muammolarni aniqlash) bandini tanlang.
  2. Adapter sozlamalarini tekshiring: Центр управления сетями → Изменение параметров адаптера bo'limida tarmoq kartasi o'chib qolmaganini tekshiring (Sifatli ishlashi uchun Включить buyrug'ini tanlang).
  3. Routerni elektr tarmog'idan 10 soniyaga uzib, qayta yoqing.
Состояние сети: Без доступа к Интернету

Tarmoq kartasi faol emas yoki xatolik yuz berdi.

12-Bo'lim. CMD orqali tarmoqni tekshirish

Tizim ma'murlari tarmoqni tekshirish va nosozliklarni izlashda Командная строка (CMD) buyruqlaridan foydalanadilar:

  • ipconfig: Mahalliy IP manzilni, tarmoq niqobini va asosiy shlyuz (gateway) manzilini ko'rsatadi.
  • ping [IP/sayt]: Tashqi server bilan bog'lanish mavjudligini va kechikish vaqtini aniqlaydi. Masalan: ping 8.8.8.8.
  • tracert [IP/sayt]: Signalning qaysi yo'nalishlar (routerlar) orqali o'tayotganini ko'rsatadi (Traceroute).
  • nslookup [sayt]: Kiritilgan saytning IP manzilini DNS serverdan so'rab aniqlaydi.

13-Bo'lim. Internet xavfsizligi

Internetga ulanish xavfsizlik choralarini bilishni talab qiladi. Ayniqsa tibbiyot muassasalarida bemorlarning shaxsiy va kardiogramma ma'lumotlari maxfiyligini saqlash shart:

Asosiy xavflar:
  • Ochiq Wi-Fi xavfi: Parolsiz bepul tarmoqlarda hackerlar ma'lumotlarni o'g'irlashi oson.
  • Phishing (Fishing): Soxta elektron xatlar orqali shifokorlar tizimiga kirish parollarini aldab olish.
Himoyalanish choralari:
  • Kuchli parollar: Router va foydalanuvchi akkauntlariga murakkab belgili parollar qo'yish.
  • VPN (Virtual Private Network): Tibbiyot muassasasi ma'lumotlarini shifrlangan maxsus xavfsiz kanal orqali uzatish.
  • Antivirus: Tarmoq orqali kiruvchi troyan viruslarini bloklash.

18-Bo'lim. Yakuniy Xulosa

Ushbu elektron darslik orqali talaba kompyuter va noutbukni internetga ulash texnologiyalari, simli (Ethernet) va simsiz (Wi-Fi) tarmoqlar farqi, Windows ruscha sozlamalari, IP va DNS manzil mantiqiy jarayoni hamda tarmoq diagnostikasi bo'yicha mustahkam amaliy ko'nikmaga ega bo'ldi.

Talabaning yakuniy AKT kompetensiyasi:
  • Lokal shifoxona tarmog'idagi Ethernet RJ-45 kabellarini to'g'ri ulash.
  • Bemorlar yoki shifokorlar noutbukida Wi-Fi parollarini kiritib internetga kirish.
  • Windows ruscha menyularida tarmoq drayverini diagnostika va restart qilish.
  • CMD komandalar satrida tarmoq sifatini to'liq tekshirish.
